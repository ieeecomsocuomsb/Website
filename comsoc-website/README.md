# IEEE Communication Society | UoM Student Branch

React + Vite site for the IEEE ComSoc student branch at the University of Moratuwa.

## Getting Started

```sh
git clone <repo-url>
cd comsoc-website
npm install
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Build the production bundle with:

```sh
npm run build
```

Preview the build locally:

```sh
npm run preview
```
